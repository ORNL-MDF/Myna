# Utility scripts

This directory contains standalone Python examples that import Myna modules directly
instead of running a workflow from a Myna input file.

These scripts assume the repository `examples/` layout. If you copy a utility script
to another location, update any example-data paths it references.

| Utility | Sample data | Optional dependency groups | External tools |
| --- | --- | --- | --- |
| `openfoam_meshing_script` | Peregrine fixture | core only | OpenFOAM |
